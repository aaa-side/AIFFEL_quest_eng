저는 간단한 코드를 작성하는데 있어 반복이 필요한 부분만 함수로 정의하여 사용하였었는데 실행하는 부분별로 함수를 나누어 사용했다는 점과

개인적으론 코드를 한두줄 더 줄일 수 있으나 가독성을 위해 남겨도 상관없을 것같습니다.(ex. num, valid_operator부분은 return과 if문에 통합하여 사용할 수 있습니다.)
